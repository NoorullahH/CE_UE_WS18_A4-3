# St. Valentin
Sankt Valentin ist die westlichste Stadtgemeinde im Mostviertel in Niederösterreich, 
genauer gesagt im Enns-Donau-Winkel und hat 9340 Einwohner (Stand 1. Jänner 2018).

## Geografie
Das Zentrum liegt auf einer Höhe von 272 Meter. Den höchsten Punkt in der Gemeinde stellt der Rohrberg da, der 400m hoch ist.

## Gemeindegliederung
Die Gemeinde St. Valentin besteht aus sechs Katastralgemeinden:
* Altenhofen
* Endholz
* Hofkirchen
* Rems
* St. Valentin
* Thurnsdorf

## Wirtschaft & Infrastruktur
In St. Valentin treffen die Donauufer- und die Rudolfsbahn auf die Westbahn, weshalb der Bahnhof ein wichtiger Verkehrsknotenpunkt ist.
Ebenfalls besteht eine Anbindung an die Autobahn A1.
